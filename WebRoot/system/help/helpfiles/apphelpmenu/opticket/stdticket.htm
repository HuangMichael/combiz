标准操作票帮助文档

在阅读此文章之前，请先阅读运行管理相关文档，以便对运行管理模块有比较全面的了解。
  1. 新建记录
  2. 保存记录
  3. 添加标准操作票步骤
  4. 删除标准操作票步骤
  5. 删除记录

1. 新建记录

请先进入运行管理->操作票>标准操作票应用程序中。

1.1 单击工具栏中的“新建记录”按钮，此时界面自动切换到如下图所示的“标准操作票”编辑界面。
　
　
　
1.2 在标准操作票编辑窗口中，可以修改系统自动生成的“操作票编号”，该编号在记录保存后将不可修改。
      其它需要填写的内容都是操作票里的项目，请依据实际情况填写或修改。

2. 保存记录

2.1 当所有底色为橙色的文本框的内容被填写完，并确保其值符合要求时，就可以点击菜单栏中如下图所示的
      “保存记录“按钮来保存本条记录。
　

3. 添加标准操作票步骤

3.1 在“标准操作票”下方的“标准操作票步骤”工具栏中，点击“添加”按钮会弹出如下图所示的
      “标准操作票步骤”的编辑界面，在这里可以为当前标准操作票中添加多条标准操作票步骤。
      这里需要填写的信息是来源于操作票中的每条操作步骤。所有这些标准操作票步骤将被作为生成
      操作票操作步骤的模板。
　
　

4. 删除标准操作票步骤

4.1 当不需要该条标准操作票步骤记录时，就可以点击“标准操作票步骤”工具栏上的中的“删除标记”

      按钮来将该条标准操作票步骤记录标记删除。真正的删除操作将在执行保存记录后执行。

5. 删除记录

5.1 当不需要该条标准操作票记录时，就可以点击工具栏上的“操作菜单”中的“删除记录”

      按钮来删除该条标准操作票记录。
