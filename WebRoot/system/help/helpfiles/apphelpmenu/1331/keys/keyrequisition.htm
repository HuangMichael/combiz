钥匙借用申请单帮助文档

在阅读此文章之前，请先阅读运行管理中钥匙台帐的相关文档，以便对运行管理模块有比较全面的了解。
  1. 新建记录 
  2. 填写钥匙借用申请单
  3. 填写钥匙借用申请行
  4. 借出钥匙
  5. 再次填写钥匙借用申请单
  6. 归还钥匙
  7. 删除记录 

1. 新建记录 

请先进入运行管理->钥匙管理->钥匙借用申请单应用程序中。

1.1 单击菜单栏中的“新建记录”按钮。
1.2 此时就新建了一张钥匙借用申请单，并自动切换到编辑界面。
 

2. 填写钥匙借用申请单

2.1 在编辑界面的“钥匙借出”功能块中可以看到借用人和借出时间已经自动生成了，可以根据实际情况修改
      该功能块中的各项参数。
2.2 填写完相应内容后点击菜单栏上面的“保存”按钮来对申请单进行保存。  
 

3. 填写钥匙借用申请行

3.1 点击“钥匙借用申请行”标签中的“添加”按钮为“钥匙借用申请单”添加一项“钥匙借用申请行”。
      “钥匙借用申请行”通俗的含义就是“钥匙借用申请单”中的要申请的一把钥匙。
3.2 此时仍然可以重复1.1步骤再添加一条“钥匙借用申请行”，也就是一张申请单要借用多把钥匙；当然也
      可以直接点击菜单栏上的“保存”按钮来保存“钥匙借用申请行”

4. 借出钥匙

4.1 借出前请先在“钥匙借用申请行”标签下方列表中将要借出的钥匙打上对勾,也就是选中一把或多把钥匙。
4.2 此时就可以点击菜单栏上的“借出钥匙”按钮将当前申请单中选中的钥匙借出，此时系统自动产生钥匙借
      用历史记录信息 。
4.3 当看到“钥匙借出成功！”提示信息时说明借出成功了，否则按相应提示信息操作。

5. 再次填写钥匙借用申请单

5.1 归还钥匙之前需要至少填写钥匙借用申请单中的“归还人”、“归还时间”两项，其余项目可不填写。

6. 归还钥匙

6.1 归还前请先在“钥匙借用申请行”标签下方列表中将要归还的钥匙打上对勾,也就是选中一把或多把钥匙。
      需要注意的是此时只能选择“钥匙借用申请行”状态是“借出”的钥匙。 
6.2 此时就可以点击菜单栏上的“归还钥匙”按钮将当前申请单中选中的钥匙归还，此时系统自动产生钥匙归
      还历史记录信息 。
6.3 当看到“钥匙归还成功！”提示信息时说明归还成功了，否则按相应提示信息操作。

7. 删除记录

7.1 请先选中一条“钥匙借用申请单”记录，双击进入该条记录的主界面。
7.2 将鼠标指针指向菜单栏中的“操作菜单”按钮，自动弹出下拉菜单，此时单击“删除记录”按钮，弹出提
      示信息“是否确定删除当前选中的记录及所有关联的子记录吗？”。
7.3 此时点“确定”则删除该“钥匙借用申请单”及“钥匙借用申请行”的记录；点击“取消”则取消操作。
